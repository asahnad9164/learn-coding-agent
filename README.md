## 🧰 Troubleshooting

### The app does not open

- Make sure the download finished
- Check that you extracted the ZIP file if needed
- Try running the file again
- Right-click the app and choose Run as administrator if your PC asks for it

### Windows blocks the app

- Open the file again
- If you see a security prompt, choose More info and then Run anyway
- Use the latest release from the releases page

### The app opens but looks blank

- Wait a few seconds
- Close the app and open it again
- Check that your display is set to a standard resolution
- Make sure no other window is covering the app

### Files are missing

- Check the save folder you chose
- Look in Documents, Desktop, or Downloads
- Search your PC for the file name
- Save again after the task finishes
